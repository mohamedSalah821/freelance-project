# 💊 El-Desouki Pharmacy Website

A modern, clean, and responsive pharmacy website built using **HTML, CSS, and JavaScript**.

Designed to deliver a professional medical experience with a focus on simplicity, trust, and usability.

---

## 🏥 Overview

This project represents a pharmacy website for:

**El-Desouki Pharmacy**  
Under the supervision of:  
- Dr. Gaber El-Desouki  
- Dr. Mohamed Gaber El-Desouki  

The goal is to create a realistic and visually appealing healthcare website suitable for portfolio presentation.

---

## ✨ Features

- 🧼 Clean and modern medical UI design  
- 📱 Fully responsive (mobile-friendly)  
- 🛒 Product cards with "Add to Cart" interaction  
- 🔔 Toast notifications instead of alerts  
- 📍 Contact section with location details  
- 💬 Floating WhatsApp button  
- ⚡ Smooth scrolling navigation  
- 🎨 Reusable UI components (cards, buttons, layout)

---

## 🧱 Tech Stack

- HTML5  
- CSS3 (Flexbox & Grid)  
- Vanilla JavaScript
